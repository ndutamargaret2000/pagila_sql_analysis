
# Pagila SQL analysis

## Overview
This project uses the Pagila dataset to perfom data cleaning and data analysis using PostgreSQL. 

The goal of this project is to answer real business questions using SQL queries.

## Business questions answered

Top customers by spending

Highest-paying customer each month

Monthly revenue growth

Running revenue totals

Top customers in each store

Customer contribution to monthly revenue

## Skills Demonstrated

- PostgreSQL
- Joins
- Aggregations
- Common Table Expressions (CTEs)
- Window Functions
- Ranking Functions
- Running Totals
- Revenue Analysis
- Customer Analysis
